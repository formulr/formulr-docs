### PDF Template Editor

A brand new editor for creating your document templates:

- **WYSIWYG Mode**: visual editor with complete toolbar
  - Formatting: bold, italic, underline, strikethrough
  - Headings H1, H2, H3
  - Bulleted and numbered lists
  - Text alignment
  - Custom font sizes
  - Page break for PDF pagination

- **Liquid Mode**: code editor with syntax highlighting
  - Variable highlighting
  - Automatic signature zone detection
  - Alerts for non-sequential signatures

- **Variables Panel**: easy insertion of data
  - Search available variables
  - Grouping by section
  - Click to insert

- **PDF Preview**: visualize the rendering with a real response

### Interactive Guide

A built-in tutorial system to discover Formulr:

- **Selection menu**: choose which guide to launch from any page
- **"Campaign Creation" tour**: first steps with Formulr
- **"Campaign Editor" tour**: sections, modules, publishing
- **Progress**: completion indicator for each guide
- **Automatic navigation**: the guide takes you to the right page

Click the **?** button in the bottom right to access the guides.
