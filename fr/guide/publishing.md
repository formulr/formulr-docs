# Publication

La publication rend votre campagne accessible aux utilisateurs. Avant de publier, assurez-vous que votre formulaire est complet et testé.

## Vérifications avant publication

Formulr vérifie automatiquement votre campagne :

### Erreurs bloquantes

Ces erreurs doivent être corrigées avant publication :

- ❌ **Sections vides** : chaque section doit contenir au moins un module
- ❌ **Modules non configurés** : tous les modules doivent avoir un libellé

### Avertissements

Ces points méritent attention mais ne bloquent pas :

- ⚠️ Aucun module obligatoire
- ⚠️ Pas de responsable assigné
- ⚠️ Pas de date butoir définie

## Publier une campagne

1. Vérifiez que votre campagne est prête
2. Cliquez sur le bouton **"Publier"** en haut à droite
3. La campagne est maintenant accessible

Le statut passe de "Brouillon" à "Publié".

## États de publication

| Statut | Description |
|--------|-------------|
| **Brouillon** | Non publié, invisible aux utilisateurs |
| **Publié** | Accessible via le lien de partage |
| **Modifications non publiées** | Publié avec des changements en attente |

## Modifier une campagne publiée

Vous pouvez modifier une campagne après publication :

1. Effectuez vos modifications dans l'éditeur
2. Le statut passe à "Modifications non publiées"
3. Cliquez sur **"Publier la mise à jour"**

::: warning Important
Les modifications sont appliquées immédiatement après publication.
Les réponses en cours verront les changements.
:::

## Dépublier une campagne

Pour retirer l'accès à une campagne :

1. Cliquez sur le menu **"Publier"**
2. Sélectionnez **"Dépublier"**
3. La campagne redevient un brouillon

Les utilisateurs ne pourront plus accéder au formulaire.
Les réponses existantes sont conservées.

## Partager la campagne

Une fois publiée, partagez votre campagne :

### Lien direct

1. Allez dans l'onglet **Paramètres**
2. Section **Partage**
3. Copiez le lien

Le lien ressemble à : `https://app.formulr.fr/c/xxxxx`

### Invitation par email

1. Dans la section Partage
2. Saisissez les adresses email
3. Validez avec Entrée
4. Cliquez sur **Envoyer**

Les destinataires reçoivent un email personnalisé avec :
- Le nom de la campagne
- Un lien direct
- Le nom de votre organisation

## Tester avant publication

::: tip Conseil
Testez votre formulaire avant de le partager largement.
:::

Options de test :

1. **Prévisualisation** : visualisez le rendu sans créer de réponse
2. **Auto-test** : remplissez vous-même le formulaire
3. **Test interne** : partagez avec un collègue d'abord

## Suivi des accès

Après publication, suivez l'activité :

- Nombre de vues
- Nombre de réponses commencées
- Nombre de réponses complétées
- Taux de complétion

Ces statistiques sont visibles dans l'onglet **Réponses**.
