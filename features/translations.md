# Translations

Formulr allows you to translate your campaigns into multiple languages to offer a localized experience to your users.

## Access Translations

1. Open an existing campaign
2. Click on the **"Translations"** tab
3. Select the target language from the dropdown menu

## Translatable Content

### General Information

- Campaign title
- Subtitle
- Description

### Step Content

For each step of your form:

- **Sections**: title and description
- **Modules**: label, placeholder, help text
- **Options**: for multiple choice fields

## Translation Interface

The interface displays side by side:

| Original | Translation |
|----------|-------------|
| Source text | Input area |

### Progress Indicators

- **Statistics**: number of translated fields / total
- **"Complete" badge**: when everything is translated
- **Filter**: show only untranslated fields

## Automatic Translation with AI

::: tip New
AI translation is available to speed up localization work.
:::

### Translate a Field

1. Click the **"Translate"** button next to a field
2. AI generates a translation instantly
3. Adjust if necessary

### Translate All

1. Click **"Translate all with AI"**
2. Confirm the action (existing translations will be replaced)
3. All translations are generated at once

::: warning Warning
AI translation replaces existing translations. Save your changes before using this feature.
:::

## Supported Languages

- French (default language)
- English
- Other languages can be added on request

## User Experience

When a user accesses your form:

1. The language is automatically detected (browser)
2. Or selected via a language selector
3. All content is displayed in the chosen language

## Best Practices

1. **Start with the main language** - Finalize the content before translating
2. **Use AI as a base** - Then refine manually
3. **Check placeholders** - They are often forgotten
4. **Test the form** - In each language before publishing
